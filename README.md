## 📋 项目简介

原洪江市黔阳一中官方网站（www.hjsqyyz.com）已无法访问（404错误）。本项目由学生自主创建，旨在存档学校历史信息，为校友及关心学校的人士提供参考。

**⚠️ 重要声明：** 本网站与洪江市黔阳一中官方无任何关联，所有信息来源于公开资料整理，仅供参考。

## ✨ 功能特性

- 📱 **响应式设计** - 完美适配手机、平板、电脑
- 🔍 **SEO优化** - 完整的搜索引擎优化，支持社交媒体分享
- ♿ **无障碍访问** - 语义化HTML，支持屏幕阅读器
- 🌙 **深色模式** - 自动适配系统深色主题
- ⚡ **纯静态** - 无需后端，加载速度快
- 🔒 **隐私友好** - 无追踪脚本，无Cookie

## 🛠️ 技术栈

- **前端：** HTML5 + CSS3 + 原生 JavaScript
- **布局：** CSS Grid + Flexbox
- **动画：** CSS3 动画 + 过渡效果
- **SEO：** Schema.org 结构化数据 + Open Graph

## 🚀 本地部署

```
# 克隆仓库
git clone https://github.com/CraftBlockht6x/hjsqyyz.git

# 进入目录
cd hjsqyyz

# 使用任意静态服务器运行
# Python 3
python -m http.server 8000

# Node.js
npx serve

# 然后访问 http://localhost:8000
```
